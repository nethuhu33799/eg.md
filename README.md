
## 1. หัวข้อ (Headings)

ใช้เครื่องหมาย `#` นำหน้า ยิ่งมี `#` มาก หัวข้อยิ่งเล็กลง (มีได้ถึง 6 ระดับ)

```markdown
# หัวข้อใหญ่สุด (H1)
## หัวข้อรอง (H2)
### หัวข้อย่อย (H3)
#### H4
##### H5
###### H6
```

ผลลัพธ์:

# หัวข้อใหญ่สุด (H1)
## หัวข้อรอง (H2)
### หัวข้อย่อย (H3)

---

## 2. การเน้นข้อความ (ตัวหนา / ตัวเอียง)

```markdown
*ตัวเอียง*  หรือ  _ตัวเอียง_
**ตัวหนา**  หรือ  __ตัวหนา__
***ตัวหนาและเอียง***
~~ขีดฆ่า~~
```

ผลลัพธ์:

*ตัวเอียง* · **ตัวหนา** · ***ตัวหนาและเอียง*** · ~~ขีดฆ่า~~

---

## 3. รายการ (Lists)

### รายการแบบไม่เรียงลำดับ (ใช้ `-`, `*`, หรือ `+`)

```markdown
- ส้ม
- กล้วย
  - กล้วยหอม   (เว้น 2 ช่องว่างเพื่อทำรายการย่อย)
  - กล้วยน้ำว้า
- มะม่วง
```

ผลลัพธ์:

- ส้ม
- กล้วย
  - กล้วยหอม
  - กล้วยน้ำว้า
- มะม่วง

### รายการแบบเรียงลำดับ (ใช้ตัวเลข)

```markdown
1. ตื่นนอน
2. แปรงฟัน
3. กินข้าว
```

ผลลัพธ์:

1. ตื่นนอน
2. แปรงฟัน
3. กินข้าว

### รายการแบบมี checkbox (Task list)

```markdown
- [x] งานที่ทำเสร็จแล้ว
- [ ] งานที่ยังไม่เสร็จ
```

ผลลัพธ์:

- [x] งานที่ทำเสร็จแล้ว
- [ ] งานที่ยังไม่เสร็จ

---

## 4. ลิงก์ และ รูปภาพ

```markdown
[ข้อความลิงก์](https://www.google.com)

![ข้อความอธิบายรูป](https://example.com/รูป.png)
```

- ลิงก์: `[ข้อความ](URL)`
- รูปภาพ: เหมือนลิงก์ แต่เติม `!` ข้างหน้า

ตัวอย่างลิงก์: [ไปที่ Google](https://www.google.com)

## 5. โค้ด (Code)

### โค้ดในบรรทัด (inline)

ใช้ backtick ` คร่อม เช่น `console.log("hi")`

```markdown
สั่งงานด้วยคำสั่ง `npm install`
```

### บล็อกโค้ด (code block)

ใช้ backtick 3 ตัว ครอบ และระบุชื่อภาษาได้ (เช่น js, python)

````markdown
```js
function hello() {
  console.log("สวัสดี Markdown");
}
```
````

ผลลัพธ์:

```js
function hello() {
  console.log("สวัสดี Markdown");
}
```

## 6. ข้อความอ้างอิง (Blockquote)

ใช้เครื่องหมาย `>` นำหน้า

```markdown
> นี่คือข้อความที่ถูกยกมาอ้างอิง
> สามารถมีหลายบรรทัดได้
```

ผลลัพธ์:

> นี่คือข้อความที่ถูกยกมาอ้างอิง
> สามารถมีหลายบรรทัดได้

---

## 7. ตาราง (Table)

```markdown
| ชื่อ      | อายุ | เมือง      |
|----------|-----|-----------|
| สมชาย    | 25  | กรุงเทพ    |
| สมหญิง   | 30  | เชียงใหม่  |
```

ผลลัพธ์:

| ชื่อ      | อายุ | เมือง      |
|----------|-----|-----------|
| สมชาย    | 25  | กรุงเทพ    |
| สมหญิง   | 30  | เชียงใหม่  |

> เคล็ดลับการจัดชิด: ใช้ `:` ในแถวเส้นคั่น
> `:---` ชิดซ้าย · `:---:` กึ่งกลาง · `---:` ชิดขวา

---

## 8. เส้นคั่น (Horizontal Rule)

ใช้ `---` หรือ `***` หรือ `___` (3 ตัวขึ้นไป) วางบนบรรทัดเดี่ยว ๆ

```markdown
---
```

---

## 9. การขึ้นบรรทัดใหม่

- เว้น **1 บรรทัดว่าง** = ขึ้นย่อหน้าใหม่
- เคาะ space **2 ครั้ง** ท้ายบรรทัด = ขึ้นบรรทัดใหม่ (ไม่เว้นย่อหน้า)

---

## 10. สรุปตารางสัญลักษณ์ที่ใช้บ่อย

| ต้องการ        | เขียนแบบนี้              |
|---------------|------------------------|
| หัวข้อ         | `# ข้อความ`             |
| ตัวหนา         | `**ข้อความ**`           |
| ตัวเอียง       | `*ข้อความ*`             |
| รายการ         | `- ข้อความ`             |
| ลิงก์          | `[ชื่อ](URL)`           |
| รูป            | `![ชื่อ](URL)`          |
| โค้ดในบรรทัด    | `` `โค้ด` ``            |
| อ้างอิง        | `> ข้อความ`             |
